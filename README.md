# The World's Worst CAPTCHA

A totally legitimate human verification system. Nothing suspicious here.

**[▶ Try it](https://maphew.github.io/worst-captcha/)**

## What is this?

It's a CAPTCHA. You click the button, prove you're human, and move on with your life. Simple.

…okay, it's a multi-stage parody CAPTCHA that puts you through an increasingly absurd gauntlet of "verification" challenges. The progress bar is *very* encouraging. Wrong answers are met with supportive affirmations. There may be easter eggs.

## Features

- 🔘 A perfectly normal "Verify" button
- 📊 A progress bar that believes in you
- 🖼️ Image recognition (sort of)
- 🎚️ Precision slider calibration
- ✅ A checkbox, because of course
- 🔢 Distorted code entry that plays fair (mostly)
- 💬 Helpful messages that get increasingly... creative
- 🥚 Easter eggs (Konami code, persistent clicking rewards, Grue ending)
- 🔊 Toggleable sound effects
- 🌙 Dark mode support
- 📱 Mobile responsive design

## Running it

It's a single `index.html` with zero dependencies. Open it in a browser.

```
open index.html
```
